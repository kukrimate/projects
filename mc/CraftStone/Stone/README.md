Stone
=====
New modding API for minecraft server!

Features will be implamented:
- Scheduled tasks
- Plugin loding from jars
- Event system
- Command system
- Help tabs
- Permission system

Building
========
1. Check out the repo with git
2. Run `gradle`
3. The compiled jar will be in the build/libs folder named STONE-API-VERSION
